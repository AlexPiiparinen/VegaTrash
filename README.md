# VegaTrash - мусор-корм для овощей

## What the fuck is that?
Не более чем чуть менее, чем полностью мусор. Простейший пример, можно управлять мурлом,
попутно оно отыгрывает анимации. Имеется баг при запуске, но достаточно начать управлять
мурлом, как он исчезнет, как будто его и не было. :-) У мурла собственный класс, да-да.

## Управление этим вашим мурлом
* Клавиша "Вверх" - движение вверх
* Клавиша "Вниз" - движение вниз
* Клавиша "Влево" - движение влево
* Клавиша "Вправо" - сам угадай.
* "ЛКМ" заставит мурло покраснеть от стыда.

## Что требуется для сборки убийцы Doom'a?
Великий и непревзайденный продукт мелкомягких Visual Studio 2010 (подойдёт любая редакция)
Библиотека SFML v.2.3.2 [для твоей любимой VS10](https://www.sfml-dev.org/files/SFML-2.3.2-windows-vc10-32-bit.zip)
